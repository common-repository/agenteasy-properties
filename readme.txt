== Description ==
<h4>Please Note: This Plugin Requires API Key from AgentEasy.com</h4>

This Plugin will install the Property Search template pages: 

1. Search Property -- display the property search
2. Search Results -- display the list of property search results
3. Property Details  -- display the property full details
4. My Active Properties - display the list of active properties of the specified agent mlsid on the plugin setting
5. My Sold Properties - display the list of sold properties of the specified agent mlsid on the plugin setting 


== Installation ==

1. Upload the plugin folder "agenteasy-properties" to "/wp-content/plugins/" directory.
1. Go to: WP Admin >> Plugins >> Activate plugin "AgentEasy Properties" 
1. Go to: WP Admin >> AgentEasy Properties >> Set the API Key.
1. Done.
